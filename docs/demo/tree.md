用清晰的层级结构展示信息，可展开或折叠。


## 基础用法

:::tip
数据扁平式结构，添加<code>v-model</code>将选择的值回传，<code>show-checkbox</code>或者<code>mutiple</code>时为多选
默认设置了<code>check-strictly</code>父子默认不关联，
新增两个属性<code>checkWithParent</code>、<code>unCheckWithchild</code>增加选择灵活性
:::
